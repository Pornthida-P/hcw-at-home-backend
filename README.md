# HCW@Home — Backend

This folder contains the **HCW@Home** backend (Sails.js). It provides the API, real-time (WebSocket), and integration logic for the HCW@Home teleconsultation platform.

HCW@Home is a fork of the HUG@Home solution co-developed by Hôpital Universitaire de Genève and Iabsis SARL. It is a scalable, institution-level secure teleconsultation system for typical telemedicine scenarios, achieved through close collaboration with healthcare professionals. The project is fully open-source and offers integrated chat, audio, and video (WebRTC).

## Features

- Voice & video consultations
- Secure chat
- Invite patient in seconds by SMS or email
- Send attachments or images
- PDF reporting
- Connect to external solutions (e.g. OpenEMR)
- Invite guest or colleagues
- External authentication (OpenID)
